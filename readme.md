> ## Como executar o projeto

para instalar as dependências, no teu console de preferência:
 ```bash
  yarn install
  ```
ou
  ```bash
  npm install
  ```

para rodar a api:
 ```bash
 yarn dev
 ```

ou para rodar com o docker-compose:

```bash
yarn docker:dev
```

---

<br />


> ## Com docker:
  Com o docker instalado e configurado no computador:
  ```bash
  yarn docker:start
  ```
